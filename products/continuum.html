About Continuum

Multispecialty, AI-assisted, anesthesia-owning.

Generic op-note tools are free-text and siloed. Continuum is a multispecialty perioperative platform with canvas-based operative drawing, anesthesia ownership, a streaming-AI documentation layer with coding and drug-interaction support, and registry submission-readiness (NSQIP, GIQuIC, AJRR, SVS VQI) — all on the shared network record.

Core Workflows

From pre-op through the full procedural arc.

01

Pre-Op Module

Pre-operative assessment documentation, surgical-safety time-out (Universal Protocol), and readiness verification before the case begins.

02

Operative Module

Intraoperative documentation with structured capture: procedure details, findings, complications, specimens, implants, and the fabric-canvas surgical drawing tool with specialty template packs.

03

Post-Op Module

Post-operative documentation, recovery notes, discharge planning, and follow-up orders. Completes the procedural arc from pre-op assessment through recovery.

04

Anesthesia record

Continuum owns the anesthesia record: Pre-op, Intra-op, and Post-op phases. ASC and other apps read it; Continuum is the single source of truth.

05

Implant & MAR panels

Implant tracking and documentation alongside the Medication Administration Record — both tied to the encounter and the operative record.

06

Operative drawing

Fabric-canvas surgical drawing tool with specialty template packs (general surgery, extensible). Draw the operative field, approach, and findings on real anatomical templates. Before/after photo capture and PDF output.

Features

The deepest AI layer across every phase.

Full procedural arc

Pre-Op, Operative, Post-Op modules covering the entire episode from assessment through recovery.

Surgical Time-Out

Universal Protocol surgical-safety checklist built into the operative workflow.

Anesthesia ownership

Single source of truth for anesthesia documentation across pre-op, intra-op, and post-op phases.

Canvas operative drawing

Fabric-canvas drawing tool with specialty anatomical templates, before/after photos, and PDF output.

Streaming AI op-note

Server-side Edge function drafts the operative note in real time as documentation progresses, with dictation cleanup.

AI coding suggestions

ICD-10-CM and CPT coding suggestions, drug-interaction checks, risk scoring with anonymized parameters.

Registry submission-readiness

Checks operative-note completeness for ACS NSQIP, GIQuIC, AJRR, and SVS VQI clinical registries.

AI surgical flags

Safety alerts, note review, and risk scoring that augments standard calculators with interaction analysis.
